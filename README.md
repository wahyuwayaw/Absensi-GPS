# Absensi-GPS
Sistem absensi modern yang memanfaatkan teknologi Face Recognition API untuk verifikasi kehadiran karyawan. Dilengkapi dengan fitur Geolocation untuk memastikan karyawan berada di lokasi yang tepat saat melakukan absensi. Sistem ini juga terintegrasi dengan modul penggajian otomatis berdasarkan data kehadiran.
